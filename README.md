# 🚗 Bot de Taxi Irapuato

## Cómo correrlo localmente

```bash
npm install
node index.js
```

## Cómo subir a Railway (gratis)

1. Sube esta carpeta a GitHub
2. Entra a railway.app
3. New Project → Deploy from GitHub
4. Agrega variable de entorno: BOT_TOKEN = tu token
5. Deploy

## Comandos del bot

- /start - Iniciar el bot
- /conductores - Ver conductores disponibles
